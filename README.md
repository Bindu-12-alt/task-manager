# ✔ Task Manager

A modern To-Do List application built with Python and Streamlit.

## 🚀 Live Demo

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://YOUR_STREAMLIT_LINK_HERE)

## ✨ Features

- Add, Edit, Delete tasks
- Mark tasks as Complete
- Priority levels (High, Medium, Low) with color indicators
- Search tasks by title or description
- Filter by All / Completed / Pending
- Data saved automatically to JSON

## 🛠️ Run Locally

```bash
git clone https://github.com/Bindu-12-alt/task-manager.git
cd task-manager
pip install -r requirements.txt
streamlit run app.py
```

## 📁 Project Structure

```
task-manager/
├── app.py            # Streamlit UI

├── requirements.txt  # Dependencies
└── tasks.json        # Local data storage
```
